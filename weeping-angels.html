Objective

Create a statue inspired by the concept of weeping angels that detects when it is not being looked at and turns to face a person in this room. While the project has little real world application on its own, aside from entertainment and decoration, the use of facial detection technology and acting based on that data opens the doors to a variety of possibilities. The success of this project was evaluated based on the reliability of the weeping angel and the success of the effect being achieved.

Technical Explanation

This project was made using Python, blender, and Visual Studio Code. For this project, there are a few clear aspects that must be considered and built. Those are the facial recognition module, the face position logger, the loss of face detection, and the servo motor to rotate the head. First, there needs to be a camera feed to find a face in. Using a small HBV-1805 camera module, a video feed of the angel’s view can be obtained. Multiple cameras can be used to increase the angel’s effective range, however using only one functions well enough that it can be used with just one.

By using python’s OpenCV image recognition library on the camera feed, the angel can find faces in its environment. For each frame the facial recognition is run, the variable lastFace can be set to the position of any face that is found. When no face is found, the variable does not change. Once there is no face detected, the loss of face detection function can send the contents of lastFace to a function that moves a servo motor to the relative position of the face in the camera view. If the facial detection picks up another face, it immediately tells the servo function to stop, so as to not ruin the illusion.

Results

Overall, the project worked perfectly and achieved the desired effect. The experience of turning away from the angel and turning back to see it staring at you works perfectly and is an effective illusion. Through this project, my understanding of facial recognition and OpenCV was reinforced and improved effectively. With these newly developed abilities, I hope to further pursue projects utilizing facial recognition in the future
